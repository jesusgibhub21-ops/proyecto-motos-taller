# Proyecto: Sistema CRUD para Taller de Motos

## Descripción del emprendimiento
Este sistema fue desarrollado para gestionar la información de un taller de motocicletas, permitiendo la administración eficiente de los datos de los clientes y sus servicios asociados.

## Problema que resuelve el sistema
El taller gestionaba la información de manera manual, lo cual generaba desorganización y dificultad al buscar registros históricos o datos de contacto. Este sistema soluciona dicho problema al centralizar la información, permitiendo realizar registros, consultas, actualizaciones y eliminaciones de datos de forma rápida y confiable.

## Tecnologías utilizadas
- Laravel 12
- PHP
- MySQL
- HTML
- CSS
- Blade

## Instrucciones para ejecutar el proyecto
1. Clonar el repositorio: `git clone [URL_DE_TU_REPOSITORIO]`
2. Instalar dependencias: `composer install`
3. Configurar el archivo `.env` con las credenciales de tu base de datos.
4. Ejecutar migraciones: `php artisan migrate`
5. Iniciar el servidor local: `php artisan serve`
5. Iniciar el servidor en visual: `composer run dev`
